Kurulum gerekli

js/firebase-config.js dosyasını aç ve Firebase projenin yapılandırma değerlerini yapıştır. Adımlar:

  1. console.firebase.google.com → yeni proje
  2. Authentication → Email/Password aktif et
  3. Firestore Database → oluştur (production modu)
  4. Firestore → Rules: FIRESTORE_RULES.txt içeriğini yapıştır
  5. Proje ayarları → Web uygulaması ekle → firebaseConfig değerlerini kopyala → js/firebase-config.js'e yapıştır
  6. Sayfayı yenile
