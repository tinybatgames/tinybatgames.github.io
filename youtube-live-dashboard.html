YouTube Data API Key

Bu dashboard YouTube Data API v3 kullaniyor. Anahtarini gir, tarayicinin localStorage'inda saklanir — sunucuya gonderilmez.

Anahtarin yoksa: console.cloud.google.com > APIs & Services > Library > YouTube Data API v3 > Enable > Credentials > Create API Key.

Demo modda da calisir — key'siz statik (kuratorluk) veriyle baslar, key girince canli moda gecer.
